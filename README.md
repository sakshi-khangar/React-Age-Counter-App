## React Age Counter App
This is a simple React practice project created using **Vite**.  
The project is mainly focused on understanding **React hooks (useState)** and **component-based structure**.

## Project Features
This project includes the following React components:

### Age Component
- Displays the user's age
- Increases age by 1 on button click
- Uses `useState` hook

### Counter Component
- Counts number of button clicks
- Updates count dynamically
- Demonstrates event handling in React

### UserInput Component
- Takes user input from textbox
- Displays typed text in real-time
- Includes reset functionality
